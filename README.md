# Country Info App

This app is build with the frontend React, Next js and backend in Nest js.

## Frontend

- React and Next.Js
- TailwindCSS
- Typescript
- Shadcn UI
- React Query
- Recharts
- React-Icons

## Backend

- Nest.js
- Typescript
- Axios

## Run the Backend

_Need to rename .env.example to .env_

```bash
cd BE
mv .env.example .env
npm install
npm run start:dev
```

## Run Frontend

_Need to rename .env.example to .env_

```bash
cd FE
mv .env.example .env
npm install
npm run dev
```
